# Test image generator

Generate N different *images* with specific *size* in a deteministic sequence from a *seed*. This is useful when testing upload images, we could easily have rememberable images to verify.

## Usage

```shell
testimgen \
  --seed \
  -s \
  --size \
  -i \
  --output \
  -o \
  N
```
